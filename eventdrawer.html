EVENT DRAWER

ROLES

Research, Branding, Design, Testing

TOOLS USED

Google Forms, Draw.io, Figma, InVision, Usability Hub

SCOPE

User Interviews, Surveys, Personas, Information Architecture, Wireframes, Prototype, User Testing

People use cloud storage products to help plan events. Google Drive to create a guestlist, Pinterest to share ideas, Dropbox to send an important PDF. Why not use a single product to perform all of these tasks? Event Drawer aims to be the one-stop-shop for event planning needs.

PROBLEM

The majority of users have a hard time predicting attendance levels and feel generally disorganized while event planning.

SOLUTION

Event Drawer is a one-stop-shop for event planning and organization. Users can create events (or drawers), upload or create related files within these drawers, invite contacts, and track attendance.

DISCOVERY & RESEARCH

I created a survey to learn more about potential problems users face when using cloud storage products. I hypothesized that people use cloud storage to help with event planning, but I needed to know if an event planning product was worth pursuing. I focused mostly on two key questions:

  1. Do people use cloud storage products to help plan events?
  2. What problems do users face when planning events?

Over 60% of respondents said they used cloud storage products to assist with event planning. The two biggest problems users faced when planning events were keeping track of attendance levels and feeling generally disorganized and overwhelmed. This survey informed me to create a product that would:

  1. Allow users to track attendance
  2. Keep all event-related info in a single, easy-to-access space

I analyzed Google Drive, Dropbox, and Pinterest, which all allow for easy file creation and collaboration. However, users would have to get creative in their use of each product to effectively use them as event planners. Why not provide a product that was specifically tailored to this purpose?
